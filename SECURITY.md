## Reporting a Vulnerability

If you discover a potential security vulnerability within any of my projects (including MYADS, Link Checker, or API integrations), please report it responsibly.

- **Private Email / Contact**: Please reach out via my official profiles on [Twitter/X (@mrghozzi)](https://twitter.com/mrghozzi) or [Facebook](https://facebook.com/mr.ghozzi).
- **Response Time**: You will receive an initial response within 24-48 hours.
- **Disclosure**: Please refrain from publicly disclosing the issue until a patch or fix has been published.

Thank you for helping keep the open-source ecosystem secure!
